# CalculaUPES
Calculadora de notas para smartphones Android. Evaluativos de Universidad Politecnica de El Salvador año 2018.

## Base de datos local
Calcula UPES dispone de una base de datos local que es administrada directamente por el usuario final, por medio de los diferentes formularios que se le presentan.

## Forma de evaluación
Calcula UPES evalua los datos requeridos de la siguiente forma:

- Nota 1 --- 30%
- Nota 2 --- 30%
- Nota 3 --- 40%
